# Tor IP Changer V3

Ultimate Tor IP rotation tool for Termux and Linux.

Features

• Automatic Tor IP rotation  
• IP information viewer  
• IP leak detection  
• Multi OS support  
• Lightweight & fast  

Supported Systems

• Termux
• Linux
• Kali Linux
• Ubuntu
• Debian
• WSL

Installation

git clone https://github.com/123tool/TOR-IP-CHANGER-V3-DARKWEB.git
cd tor-ip-changer-v3-darkweb
chmod +x install.sh
bash install.sh

Usage

bash ~/.tor-ip-changer/tor-ip-changer.sh
